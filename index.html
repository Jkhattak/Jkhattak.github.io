Jawad Khattak

"All models are wrong, but some are useful." — George Box

I'm a data analyst at Public Interest Registry, the nonprofit behind .org. My work tends to span well beyond the title. I've built and deployed a domain renewal prediction model using XGBoost that runs on a monthly production cadence, built a nonprofit database from scratch using Lambda functions, web scraping, and Snowflake pipelines, and done work on registrant behavior to understand who is actually buying domains and why. Currently building a Claude MCP integration for Tableau. Finishing an MS in Computer Science at Georgia Tech, concentrating in machine learning.
