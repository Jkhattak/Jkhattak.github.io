// about

Data analyst at Public Interest Registry, a nonprofit that manages the .org top-level domain. The work spans more than the title suggests. I've built a unified nonprofit database from scratch using Lambda functions, web scraping, Snowflake pipelines and streams, joined against internal datasets to surface insights about how .org contributes to nonprofit sector growth across the US and UK. I trained and deployed a first-year domain renewal prediction model using XGBoost, set up a monthly production cadence, and fed those outputs into broader business decisions. I've also done work on registrant behavior to understand what types of buyers sit behind each purchase. Pursuing an MS in Computer Science with a concentration in machine learning at Georgia Tech.
